# time-decision-log

**素早い意思決定と日々の行動記録**のための、軽量なWebアプリケーション。

迷う時間を減らし、選択肢をクリックするだけで  
「今、何をしているか」を即座に記録できます。  
アプリは自動でタイムスタンプを保存し、**日次・月次・四半期**のサマリーを  
1ページにまとめて表示します。
このアプリは、
フロントエンドが中心のアプリです。【フロントエンドは、人が見る画面を作るところです。】
画面の部分は自分で作って、
データの保存は、Supabaseという外部サービスを使っています。【バックエンドは、データを保存するところです。Supabase：データをあずける外の箱】

---

## ✨ 機能（Features）

- ✅ **ワンクリック行動ログ**  
  あらかじめ用意された項目、または自分で追加した項目をクリックするだけで、
  活動を即時記録できます。

- 📊 **自動統計**
  - 日別サマリー
  - 月別サマリー
  - 四半期サマリー
  - カテゴリー別の回数・割合

- 🧾 **生ログ一覧（Raw log table）**
  - タイムスタンプ付きですべての記録を表示
  - 必要に応じて個別に削除可能

- 🗂 **カスタムカテゴリー**
  - 学習、コーディング、仕事、休憩など、自由に項目を追加可能
  - カテゴリーはローカルに保存されます

- 💾 **ローカルストレージ**
  - データはブラウザの `localStorage` に保存
  - バックエンド不要、アカウント不要、データ送信なし

- 📤 **CSVエクスポート**
  - 全記録をCSV形式で出力し、外部分析が可能

---

## 🧠 目的（Purpose）

このプロジェクトは、次のシンプルな問題を解決するために作られました。

> **考えすぎて時間を無駄にしない。  
> 行動し、記録し、振り返るだけ。**

行動記録のコストを極限まで下げることで、以下を実現します：
- 自己認識の向上
- 意思決定疲れの軽減
- 実際に時間がどう使われているかの可視化

## 👩 テスター①（一般ユーザー視点）

**背景：家族（姉）**
- 小学校の数学教師として **約6年間の指導経験**
- IT専門職ではない立場
- 日常的に「分かりやすく伝える」ことを重視する職業背景

**主なフィードバック：**
- 画面だけでは **アプリの目的や使い方が分かりにくい**
- 用語や表現がやや抽象的で、一般ユーザーには理解しづらい
- 機能の意味が直感的に伝わらない

**改善方針：**
- 画面内に **分かりやすい説明文（UIガイド）** を追加
- 専門用語を避け、**誰でも理解できる表現**に変更
- 各機能に **補足説明（ツールチップ／説明エリア）** を追加
- 初回利用者向けの **簡単な操作説明（オンボーディング）** を検討

## 👨 テスター②（技術・実務視点）

**背景：家族（義兄）**
- IT業界で **約10〜12年の実務経験**
- 初期キャリア：
  - **約3年間：Javaによるバックエンド開発**
- 近年：
  - **約7〜8年間：企業向けBIプロダクトの運用開発**
  - **高度なテクニカルサポート業務**を担当

**主なフィードバック：**
- スマートフォン表示時に **画面幅が端末に最適化されていない**

- ボタンサイズ・余白の最適化

## 🧠 開発者視点での追加要件【LI NA】

- **①多言語対応（Internationalization）**  
  日本語 / 中国語などの多言語表示切替機能
  文言を一元管理し、拡張しやすい構成にする
- **②マルチユーザー・ロール管理**  
  複数ユーザーを同一アカウント内で管理
  例：管理者（会社代表）
     一般ユーザー（従業員）
  ユーザーごとに記録を選択・切替・集計できる仕組み
- **③モバイル用クイック操作**  
  ホーム画面へのショートカット追加、操作の高速化
- **④データ同期**  
  PC・スマートフォン間のリアルタイム同期
- **⑤クイック記録の管理機能**  
  登録済みのクイック操作に対して：
                         削除機能
                         編集機能
  ユーザーが自由に管理できるUI
- **⑥状態の時間計測ロジック**  
  状態ボタンを1回クリックすると「開始」
  次の状態をクリックするまで、同一状態が継続
  各状態の滞在時間を自動計測
👉 時間ベースの行動分析を可能にする設計
- **⑦期間指定と集計**  
  当日／当月／当四半期のデフォルト表示＋任意期間指定
  デフォルト表示：
             当日：当日の24時間
             当月：現在の月
             当四半期：現在の四半期
  任意期間指定：
            開始日（年月日）
            終了日（年月日）
- **⑧統計・可視化**  
  回数・割合・時間・時間割合の集計、リアルタイム更新の階層表示
  カテゴリ別集計：
             回数
             回数比率（％）
             合計時間
             時間比率（％）
  リアルタイム更新されるツリー構造（階層表示）
  当日の行動状況を即時把握できる可視化
- **⑨ 生成AIを活用した対話型データ分析（AI活用視点）**

  ユーザーが自然言語で質問できる  
  **対話型検索UI** を想定しています。

  AIは、保存されている  
  **すべての行動履歴・時間データ** を参照し、  
  質問の意図を理解したうえで、  
  自動的に **集計・要約** を行います。

  **質問例：**
  - 「AI・ITの学習に最も時間を使った日はいつか」
  - 「今月、一番集中していたカテゴリは何か」

  数値やグラフを見るだけでなく、  
  **対話を通じて行動を振り返る体験** を提供することを  
  目的としています。
